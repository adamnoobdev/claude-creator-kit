10 sekunder. Animationen ovan är renderad med hyperframes-skillen från kit:et.

Vad det är

En mapp med verktyg du kan använda från Claude direkt i din kodeditor. Du säger vad du vill göra på svenska, Claude använder verktygen och levererar filerna. Ingen kod du behöver lära dig. Bara konversation.

Vad du kan göra

VID

Klippa video

Transkribera tal, klippa ihop highlights, lägga på subtitles. Allt via samtal.

HF

Animationer

Rendera HTML och GSAP till MP4 i valfri dimension. Text-cards, lower-thirds, charts, CTA-overlays.

IMG

Generera bilder

Nano Banana 2 via Kie.ai. Stills, portraits, product-shots på 10-30 sek per bild.

SFX

Musik och sfx

ElevenLabs för bakgrundsmusik och ljudeffekter. Beskriv genre och mood, få fil.

Kom igång

  1. 1

    Installera VS Code

    Microsofts gratis kodeditor. Det är miljön du jobbar i.

    Ladda ner VS Code →
  2. 2

    Lägg till Claude-extension

    I VS Code: öppna Extensions (Cmd+Shift+X), sök "Claude Code", klicka Install. Logga in när du blir promptad.

  3. 3

    Installera Node, FFmpeg och Python

    Behövs för att verktygen ska kunna rendera video och anropa API:er. På Mac, i Terminal:

    brew install node ffmpeg python

    Saknar du Homebrew? Installera från brew.sh först.

  4. 4

    Ladda ner kit:et

    Klona via terminal eller ladda ner som ZIP från GitHub.

    git clone https://github.com/adamnoobdev/claude-creator-kit.git
    Ladda ner som ZIP →
  5. 5

    Fyll i dina API-nycklar

    Öppna mappen i VS Code. Kopiera .env.example till .env och klistra in nycklar från tjänsterna nedan.

  6. 6

    Säg till Claude: "Hjälp mig komma igång"

    Öppna Claude-panelen i VS Code och skriv det. Claude tar det därifrån, frågar vad du vill skapa och scaffoldar ditt första projekt.

Konton du behöver skapa

Alla har gratis-tier som räcker för att testa.

  • Anthropic Claude API för automations utöver chatten.
  • Kie.ai Bildgenerering med Nano Banana 2.
  • ElevenLabs Musik och ljudeffekter.

Andra verktyg värda att kolla in

Inte del av kit:et, men funkar utmärkt vid sidan av Claude i ett creator-flöde.

  • Magnific AI-uppskalning och bildförbättring. Höjer resolution och detaljnivå på AI-genererade bilder eller foton. Ägs nu av Freepik.
  • Higgsfield AI-videogenerering med kamerakontroll. Cinematiska clips, motion graphics och karaktärs-animationer från text eller bild.
  • Wispr Flow Dictera istället för att skriva. Pratar du, så transkriberar appen direkt i vilken text-input som helst. Sparar tid på långa prompts.
