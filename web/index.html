Claude Creator Kit

Producera video, bild och ljud snabbare med Claude Code.

Vad är det?

En öppen startpunkt för creators som vill använda AI på riktigt i sitt produktionsflöde. Du klonar en mapp, fyller i tre API-nycklar, och har verktyg för att klippa video, generera bilder, skapa animationer och göra musik. Allt drivet via samtal med Claude Code i terminalen.

Inget hemligt eller magiskt. Bara en organiserad mapp och några små skripts.

Kom igång

  1. Installera Claude Code. Följ instruktionerna på claude.com/code.
  2. Installera Node, FFmpeg och Python. På Mac: 
    brew install node ffmpeg python
  3. Klona kit:et. 
    git clone https://github.com/adamnoobdev/claude-creator-kit.git
cd claude-creator-kit
  4. Fyll i dina API-nycklar. 
    cp .env.example .env
    Öppna .env i en editor och klistra in nycklar från tjänsterna nedan.
  5. Starta Claude Code. 
    claude
    Säg sen till Claude: "Hjälp mig komma igång".

Vad du behöver registrera dig för

  • Anthropic Console För Claude API. Gratis-tier ger dig credits att testa med.
  • Kie.ai För bildgenerering med Nano Banana 2. Gratis-tier finns.
  • ElevenLabs För musik och ljudeffekter. Gratis-tier med månads-credits.

Vad du kan göra

Klipp video

Be Claude transkribera, klippa, lägga subtitles på dina filer. Du säger vad du vill, Claude gör det.

Bygg 9:16-animationer

HyperFrames-skillen renderar HTML+GSAP till MP4. Bra för text-overlays, charts och CTA-kort till Reels och TikTok.

Generera bilder

Nano Banana 2 via Kie.ai. Stills, portraits, product-shots, backgrounds. Beskriv vad du vill, få bild på 10-30 sek.

Skapa musik och ljud

ElevenLabs Music för bakgrundsmusik, Sound Effects för korta sfx. Beskriv genre och mood (men inga artistnamn).

Klart att börja?

Kopiera kommandot och kör i din terminal.
